# Contacts App

## Run Locally 

Clone the project  

~~~bash  
git clone https://github.com/sulekaptan/contacts-app.git
~~~

Go to the project directory  

~~~bash  
cd contacts-app
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the project  

~~~bash  
npm start
~~~
